# Магазин мебели
Данный проект является учебным и выполнен для закрепления изученного материала.
Проект вы полнен на чистом JS, HTML и SASS. Адаптивнось реализована с попощью медиа запросов.При именовании классов соблюдена БЭМ-методология. Реализована доступность для взаимодействия с клавиатуры и для голосовых читалок с использованием aria атрибутов. Анимация реализована с использованием keyframes. Реализована сборка проекта с помощью gulp. Часть функцонала реализована с помощью сторонних библиотек:
 * swiper - для реализации слайдеров;
 * Choices-js - для реализации пользовательского селекта;
 * tippy.js - для реализации tooltipe;
 * just-validate - для валидации полей формы;
 * inputmask - для маскирования полей формы.


## Запуск проекта
1. В корне проета необходимо в терминале выпонить команду `npm i`.
2. Для сборки в корневой папке проекта необходимо в терминале выпонить команду `npm run build`.
3. Открыть на локальном сервере файл index.html из папки dist.

## Описание функционала проекта
1. Проект представляет собой страницу интернет-магизмна мебели.
2. Реализован адаптив и доступность для взаимодействия с сайтом через клавиатуру и для голосовых читалок. При изменнии размера окна через devTools необходимо перезагружать страницу. Свёрстан был  PixelPerfect.
3. Реализованы 3 страницы:
 * Главная;
 * Подробного описания товаров. На неё можно попасть кликнув на ссылку "Купить";
 * Каталога товаров. На неё можно попасть кликнув на ссылку "Каталог";
4. Реализован переиспользуемый блок со слайдером.

### Главная страница

1. В header реализован выпадающий список с выбором региона и категорий товара.
2. Реализован слайдер с индикаццией времение переключения слайда (для десктопа).
3. В разделах "Специальные предложения", "Полезное"  реализован слайдер.
4. Реазизована валидация формы заказа обратной связи.


### Страница подробного описания товар
1. При нажатии на маленькие изображения, меняется большое.
2. При нажатии на большое изображения товара открывается окно подробным просмотром фото товара.
3. Реализован слайдер в разделе "Похожие товары".

### Страница каталога
1. Реализован "ползунок" фильтра цены.
